# RPA Challenge Automation

This project automates the RPA Challenge using Python and Selenium.

https://rpachallenge.com/

## Installation
```bash
git clone https://github.com/QG-dude/rpa-challenge.git
cd rpa-challenge
pip install -r requirements.txt
```
